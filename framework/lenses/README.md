## レンズの追加

新しいレンズを追加するには、このディレクトリに `<レンズ名>.md` を置き、
本ファイルの一覧に行を追加するだけ。
レンズは「常時適用ではない」という性質上、軸（axes）よりも気軽に追加・削除してよい。
