# 地域ペルソナ: 欧州

> 親: [[../README|personas]] | 兄弟（地域）: [[north-america|北米]] / [[china|中国]] / [[japan|日本]] / [[southeast-asia|東南アジア]]

## 通信インフラの特徴

- 規制主導の市場（GDPR、デジタルサービス法）
- MNO の収益性が低く、統合の圧力
- エネルギーコストが高い（特にウクライナ危機以降）
- Sub-6GHz 中心の展開、ミリ波は限定的

## 主要プレイヤー

- **MNO**: Deutsche Telekom, Orange, Vodafone, Telefonica
- **ベンダー**: Ericsson, Nokia
- **政策**: EC, ETSI, 6G-IA, Hexa-X プロジェクト

## ペインポイント

- ネットワークのエネルギーコスト（最大の関心事）
- 5G の収益化の遅れ
- 中国ベンダー排除後の代替確保

## 技術への関心

- エネルギー効率が最優先（Green Deal との連動）
- RIS（低電力カバレッジ拡張）
- Semantic Communications（帯域効率向上）
- サステナビリティ KPI の標準化

## この地域が重視する価値軸

- エネルギー効率（最重要）
- 信頼性（産業用途、Industry 4.0）
- カバレッジ（農村部、特に北欧）
