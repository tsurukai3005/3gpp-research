# 30_meetings — 会合スナップショット

RAN1 会合ごとの時点情報。Git log 的な「この会合でこう動いた」の記録。

## ディレクトリ構造

```
30_meetings/
└── ran1-NNN/
    ├── overview.md    ← 会合全体の読解メモ
    └── ffs.md         ← 残された FFS 項目（特許創出の最短距離）
```

## FFS の重要性

Chairman's Notes の FFS（For Further Study）項目は、
次回会合で解決策が求められる課題。ここに向けた解決策が特許のコアとなる。
（戦略文書 5.3 節参照）
