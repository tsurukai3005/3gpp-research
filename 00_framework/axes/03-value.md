# 軸3: 価値軸

**問い: この技術はどの KPI を改善し、何を犠牲にするか？**

## KPI 分類

| KPI | 説明 | 6G での重要度 |
|:---|:---|:---|
| スループット（peak/cell-edge） | データ伝送速度 | 中（もはや主目的ではない） |
| 遅延 | E2E レイテンシ | 高（URLLC の進化） |
| 接続数 | 同時接続端末数 | 高（IoT 拡大） |
| **エネルギー効率** | ネットワーク/端末の消費電力 | **最高**（OPEX 削減直結） |
| **カバレッジ** | 通信可能エリアの広さ | **最高**（NTN、過疎地） |
| 信頼性 | パケット到達率 | 高（産業用途） |
| **AI 統合度** | AI/ML の活用深度 | **最高**（6G の差別化要素） |
