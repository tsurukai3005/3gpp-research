\begin{frame}{SUL の設計動機 ─ 高周波 TDD の UL カバレッジ問題}
\begin{itemize}
  \item C バンド (n78, 3.5 GHz): 広帯域だがパスロス大
  \item DL: gNB は高出力 → カバレッジ確保可能
  \item UL: UE 送信電力上限 23 dBm → カバレッジボトルネック
  \item SUL の解決策: 低周波帯 (n80/81/82) を UL 専用に追加
  \item 結果: DL = C バンド広帯域、UL = 低周波で良好なカバレッジ
\end{itemize}
\end{frame}
