\begin{frame}{Section 5.3.4 ─ RB 整列と複数ニューメロロジー多重化}
\begin{itemize}
  \item 5.3.4: 異 SCS の BWP が同一キャリア内で共存する際の RB 整列規則
  \item 30 kHz の RB 境界 = 15 kHz RB 2 個分にネスト → CRB ナンバリングの一貫性
  \item 異 SCS 間の問題: OFDM 直交性が崩れる → INI (Inter-Numerology Interference)
  \item 仕様上は BWP 間に明示的なガードバンド要件なし → 対策は実装判断
  \item 実装上の対策: BWP 間の周波数的分離 / ウィンドウイング・フィルタリング
\end{itemize}
\end{frame}
