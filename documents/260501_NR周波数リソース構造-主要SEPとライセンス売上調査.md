# NR 周波数リソース構造 — 各社主要 SEP とライセンス売上の調査

> 親: [[260421_NR周波数リソース構造の発表資料ドラフト|発表資料ドラフト]] / 関連: [[260420_標準化特許申請のための調査戦略|SEP 戦略]] / [[260424_Rel-15-企業別戦略背景と対立候補波形|Rel-15 企業別戦略]]

## 0. 調査スコープと前提

### 対象範囲
発表資料ドラフトのスコープ（TS 38.101-1 Section 5 + TS 38.211 Section 4）に登場する以下の概念群:

| カテゴリ | 概念 | 仕様セクション |
|:--|:--|:--|
| 帯域定義 | Operating Band, FDD/TDD/SDL/**SUL** | 38.101-1 §5.2 |
| 帯域コンビ | Band Combination, **CA**/DC | 38.101-1 §5.2A-D |
| 帯域幅 | Channel BW, N_RB, Guardband | 38.101-1 §5.3 |
| ニューメロロジー | **SCS**, Mixed Numerology | 38.211 §4.2 |
| ラスター | Channel Raster (NR-ARFCN), **Sync Raster (GSCN)** | 38.101-1 §5.4 |
| デュプレックス | TX-RX Frequency Separation | 38.101-1 §5.5 |
| 内部分割 | CRB, **BWP**, PRB, Point A | 38.211 §4.4 |
| 集約 | **CA** (物理層), ca-SlotOffset | 38.211 §4.5 |

**太字** = 特許密集領域（後述）。

### 公開情報からの取得限界

本ノートは **"各特許単位"** ではなく **"各社の累積セルラー IPR ライセンス売上（3G+4G+5G まとめ）"** を整理する。理由は以下の通り:

1. **SEP は単一特許単独でライセンスされない**: FRAND コミットメントの下、各社は **ポートフォリオ単位**でクロスライセンス・FRAND 料率を設定する。「特許 X の売上は Y ドル」という会計区分は存在しない。
2. **"標準必須性" は宣言ベース**: ETSI に declare された SEP の多数は、訴訟や独立第三者の精査で必須でないと判定されることがある（"over-declaration" 問題）。Cyber Creative Institute / IPlytics の調査で declared / true essential の比率は概ね 30–50% と報告されている `[要確認]`
3. **ランキングの方法論依存性**: declared family count（IPlytics）／granted family count／contribution rate / Patently Score／forward citation など指標で順位は大きく変動する。**単一の "5G No.1" は存在しない**。
4. **会計上の開示有無の偏り**:
   - 開示企業（純ライセンサー寄り）: Qualcomm（QTL）／Nokia（Nokia Technologies）／Ericsson（IPR licensing）／InterDigital／Huawei（任意開示）
   - 非開示企業（実装者兼権利者）: Samsung／LG／ZTE／Apple／Sharp／Xiaomi／OPPO／NTT DOCOMO はライセンス売上を区分開示していない（クロスライセンスで相殺される傾向）

このノートは特許助言・法的助言ではない（[`framework/skill-contract.md`](../framework/skill-contract.md) §Global Will Not 6）。**研究目的での公開情報サマリ**である。

## 3. 周波数リソース構造領域における主要 SEP（候補）

> **前提**: 以下は patents.google.com / Justia / USPTO 等での公開特許検索でヒットした **代表例**。3GPP 仕様書の特定条文と必須性が一致するかの精査（claim chart）は本調査の範囲外。**"主要"** とは 「公開情報で頻繁に参照される」「assignee が Top 6 のいずれか」「3GPP の主要機能と直接関連する」 を意味する。

### 3.1 BWP（Bandwidth Part）

|:--|:--|:--|:--|
| US10735078B2 | Operations with bandwidth part (BWP) switching | Qualcomm | TS 38.211 §4.4.5, TS 38.213 BWP switching |
| US20230072549A1 | BWP partition and sub-BWP hopping | Qualcomm | （Rel-17 以降の細分化） |
| US20190274162A1 | BWP configuration for subband access in NR-U | Qualcomm | TS 37.213 (NR-U) |

| US20210127359A1 | BWP for unicast/multicast and resource allocation for multicast | （要確認） | MBS / Rel-17 |

BWP は Rel-15 で新規導入された概念で、`bwp-InactivityTimer`（タイマー）／`DCI Format 0_1/1_1` での切替／`offsetToPointA` 連携など、Qualcomm が早期に出願を集中させている領域 `[要確認: Samsung/Huawei/LG/NTT DOCOMO の同等出願件数]`

### 3.2 SUL（Supplementary Uplink）

公開検索ではヒット数が他より少なく、寄書ベースで主導した企業を辿る方が確度が高い:

- Qualcomm — `RWS-180008` (RAN Workshop 2018) で SUL を含む NR PHY をプッシュ。Switched Uplink white paper (2023) で運用面の深掘り
- Ericsson — Ericsson + STC + Qualcomm の 4CC mmWave UL 共同開発 (2023) は SUL の発展形
- Huawei — 中国オペレータ向けに SUL で n80/n81 を活用する基地局実装ノウハウ `[要確認]`

SUL は基地局スケジューラの実装ノウハウ寄りで、純粋な "アルゴリズム特許" よりも "シグナリング手順 / 上下リンク切替判定基準" 系のクレームが中心。SEP 取得には RAN1 + RAN2 にまたがる視点が必要。

### 3.3 Mixed Numerology / Carrier Aggregation

| 公開番号 | タイトル | Assignee | 仕様対応 |
|:--|:--|:--|:--|
| US11032036B2 | Carrier aggregation (CA) of mixed numerologies | （要確認、検索結果でヒット） | TS 38.211 §4.5, ca-SlotOffset |

| 公開番号 | タイトル | Assignee | 仕様対応 |

### 3.5 TX-RX Frequency Separation / 逆デュプレックス（n20, n71）

純粋な周波数値の規定は SEP にならないが、関連領域として:
- 自己干渉キャンセラ（IBFD: In-Band Full Duplex）: Rel-18 以降の主要 SI で Qualcomm / Samsung / LG が出願集中 `[要確認]`
- フィルタ実装（デュプレクサ）: Skyworks / Qorvo / Murata 等の RF コンポーネントベンダーが厚い（3GPP SEP より部品 IP の領域）

## 5. "影響範囲の大きい主要" SEP の絞り込み

### 5.1 影響範囲（インパクト）の評価軸
1. **必須性**: その特許を回避すると 3GPP 規格に従う製品が作れないか
2. **市場露出**: 該当機能を実装する製品台数 × ASP（市場規模）
3. **訴訟参照頻度**: 過去の SEP 紛争で該当特許が引用された件数
4. **declared/granted 比**: 宣言だけでなく権利化されているか

### 5.2 本発表スコープでの暫定 "重要 SEP 領域" Top 5（個別特許ではなく機能領域）

| 領域 | 影響範囲 | 主導企業（推定） | 根拠 |
|:--|:--|:--|:--|
| **BWP 切替（DCI / タイマー / RRC）** | 高（全 NR UE が実装） | Qualcomm 主導、Samsung / Huawei / LG 追随 | Rel-15 新規概念、初期出願集中 |
| **異 SCS 多重化 / CRB 整列 / ca-SlotOffset** | 高（CA を使う全 NR UE） | Qualcomm / Ericsson / Huawei / NTT DOCOMO | LTE-A CA 資産の継承 |
| **GSCN ベースの2段階セルサーチ** | 高（全 NR UE 必須） | LG / Samsung / Qualcomm `[要確認]` | LTE 既存 → NR で粗ラスター追加 |
| **SUL 動的切替 / UL カバレッジ補助** | 中（n78 + n80 系運用で発火） | Qualcomm / Ericsson | 中国オペレータの C+L バンド運用で実装顕在化 |
| **kSSB / offsetToPointA / Half-SC offset** | 中（実装上の調整パラメータ） | RAN4 主導企業（Ericsson / Nokia / Huawei） | 38.101-1 §5.4.2.2 の半 SC 許容 |

### 5.3 Top 5 の "影響範囲の大きさ" の根拠
- **BWP**: 100% の NR UE が実装（Initial BWP は MIB/CORESET#0 から導出される必須機能）。RedCap UE まで含めると累計出荷台数 数十億規模 `[要確認]`
- **CA + 異 SCS**: Cat-12 以上の高位 UE で必須。eMBB / FWA で常用
- **GSCN セルサーチ**: 電源 ON / セル再選択ごとに発火 → 端末ライフタイムで毎日数十回
- **SUL**: 中国 / 一部欧州オペレータでのみ展開 → 影響は限定的だが C-band 5G 普及で今後拡大
- **kSSB / Point A**: 全セル設定で毎回参照される設計パラメータ

## 6. 前世代との差分（4G LTE-A → 5G NR）

[260420_NRフレーム構造とリソースブロックの進化まとめ]] の周波数領域版補足:

| 概念 | LTE / LTE-A | 5G NR | SEP 創出機会 |
|:--|:--|:--|:--|
| キャリア帯域幅 | 最大 20 MHz | 最大 100 MHz (FR1) / 400 MHz (FR2) | ほぼ消化済み |
| ニューメロロジー | 15 kHz 固定 | 15/30/60/120 kHz（μ ベース） | **大** — Mixed Numerology 共存 |
| BWP | なし（UE は常時全帯域モニタ） | **新規** Rel-15 | **大** — Qualcomm が先行 |
| CA | LTE-A Rel-10 で導入 | NR では最大 16+16 CC | 中 — NR 固有拡張のみ |
| Sync Raster | 100 kHz 単一ラスター | 2層（Channel + Sync） | 中 — 高速セルサーチ |
| SUL | LTE では限定的（Bandwidth class） | **新規** Rel-15 | 中 — 中国オペレータ駆動 |

**進歩性の源泉**: LTE は単一 SCS / 単一 BWP 概念がない / 帯域幅も狭く、 **5G で解決すべき "100 MHz 帯域での UE 省電力 と 異サービス共存" こそ周波数領域 SEP の進歩性源** である（[`framework/principles.md`](../framework/principles.md) §2 の実装制約ギャップ思考）。

### 7.3 戦略的な含意（プロジェクト目的への接続）
本プロジェクトは「**新規 SEP の創出**」が目的（[`CLAUDE.md`](../CLAUDE.md) §プロジェクトの目的）。本調査の含意は:

1. **Rel-15 周波数リソース領域は既に主要 SEP がほぼ確立済み**（特に BWP / CA / Mixed Numerology）。後発で同じ概念での進歩性は厳しい
2. **新規 SEP の余地は次の領域**:
   - Rel-18/19 RedCap での BWP 拡張（最大 20 MHz 制約下の動作）
   - 6G での Mixed Numerology 拡張（cmWave / sub-THz での新 SCS 候補, μ=4,5,6）
   - NTN 用 BWP / SUL（LEO 衛星向けドップラ補正 BWP 構造）
   - AI 駆動 BWP 切替予測（Rel-19 AI/ML for Air Interface との接続）
3. **既存 Top 6 が押さえていない狭間**（学術論文の進歩性 vs 3GPP 実装制約のギャップ）を [[260421_フレーム構造のギャップ分析-学術vs3GPPvs実装制約]] で深掘りする価値が高い
4. **declared 件数だけでなく essentiality rate（精査後の必須率）** を意識した出願文化が、長期的なポートフォリオ価値を決める。サイバー創研系の精査ランキングで上位に残る企業の出願スタイルを参考にすべき

## 8. サマリー（発表資料の Q&A 用）

> 「BWP や CA はどの会社の特許で押さえられているのか?」と質問された場合の **30 秒回答**:

- 5G SEP 全体は年間約 **$15B** の市場で、Top 6（Huawei / Samsung / Qualcomm / LG / Nokia / Ericsson）が **約65%** を declared 占有
- 純ライセンサーの **公表セルラー IPR 売上（2024）** は Qualcomm $5.57B → Nokia $2B → Ericsson $1.5B → InterDigital $0.87B → Huawei $0.63B
- **declared 件数と精査後の必須性ランキングは順位が異なる**:
  - declared (IPlytics): Huawei → Samsung → Qualcomm → ...
  - 精査後 (サイバー創研 2024-05): Huawei 10.0% → Samsung 9.7% → NTT DOCOMO 7.8% → LG 7.7% → Qualcomm 7.6% → ZTE → Ericsson → Nokia → ...
- BWP は Rel-15 新規概念で **Qualcomm が先行出願集中**、その後 Samsung / Huawei / LG / NTT DOCOMO が追随
- CA / 異 SCS 多重化は LTE-A 時代からの蓄積で **Qualcomm / Ericsson / Huawei / Samsung / LG / NTT DOCOMO** が分散保有
- 個別特許の売上は **原理的に開示されない**（ポートフォリオ FRAND ライセンスのため）。「BWP の特許で年間いくら」は答えられない問い
- 自動車では **Avanci 5G プール（$29-32/車）** で集約 — 80% の 5G SEP をカバー
